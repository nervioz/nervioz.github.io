Все манипуляции нужно делать в браузере телефона
В поле sbpId указывается идентифкатор банка в виде bankXXXXXXXXXXX, это sbp memeber id банка XXXXXXXXXXX, по умолчанию для бифита sbpId = bank600000000001
В поле tokenIntentId указывается идентификатор намерения привязки счета, для теста может быть любой строкой, ни на что не влияет
По кнопке Generate будет ссылка по нажатию должно открыться приложение
